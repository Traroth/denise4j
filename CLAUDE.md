## Project Overview

`denise4j` est une bibliothèque graphique Java inspirée des effets demoscene des années 80/90. Elle exploite les classes d'images Java (`BufferedImage`, etc.) pour produire des effets et animations graphiques.

**Domaine :** scrolling, zoom, rotation, déformation, changement de palette, compositing d'images.

- **groupId:** `fr.dufrenoy.imagefx`
- **artifactId:** `denise4j`

## Commands

```bash
# Compiler
mvn clean install

# Lancer les tests
mvn test
```
